最近在学习python的标准版，发现《Python 3 Module of the Week》这个系列不错，计划边学习边翻译出来。
PyMOTW-3 是[Doug Hellmann](https://doughellmann.com/blog/)写的一系列关于如何使用Python3标准库的示例文章，基于之前的python2.7的[PyMOTW](https://pymotw.com/2/)系列。
